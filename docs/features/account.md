# Account

## Authentication Modes

### Local Account (default)
- Works without any registration.
- Data is stored on-device only.
- Data is lost on uninstall or device switch.
- Lowest barrier to entry.

### Phone Authentication
- Authenticated via phone number (OTP).
- Phone number is not stored after authentication.
- Enables cloud sync across devices.
- Data is preserved on reinstall or device switch.

## Account Data

- Display name, description, avatar.
- Avatar can be uploaded and changed.
- User Share ID: 6-digit code for sharing the account profile with others.
- Account data is cached locally for offline access.
