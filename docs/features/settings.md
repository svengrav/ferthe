# Settings

## Theme

- Switch between Dark and Light mode.
- Preference is persisted and applied on app restart.
- Affects all UI components.

## Language

- Switch between English (EN) and German (DE).
- Preference is persisted.
- All text, labels, and messages update immediately on change.

## Persistence

- Settings are stored locally on-device.
- Synced across devices when using phone authentication.
